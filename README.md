<h1>Backend para seleção de desenvolvedor full-stack Shopper</h1>

<h2>Como usar:</h2>
<li>1-Git Clone</li>
<li>2- Cd backend</li>
<li>3- npm install</li>
<li>4- criar .env</li>
<li>5 - npm run migrations</li>
<li>6 -npm run dev</li>
